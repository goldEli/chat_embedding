# chat_embedding 

聊天机器人嵌入脚本

 [![NPM version][npm-image]][npm-url] 


[npm-image]: https://img.shields.io/npm/v/chat_embedding.svg?style=flat-square

## Getting started
### Install

```shell
npm install chat_embedding
```

### How To Use

```JavaScript
import chatEmbedding from "chat_embedding"

chatEmbedding.run({
  // 聊天机器人服务地址
  serverUrl: "https://www.baidu.com/",
})

```
