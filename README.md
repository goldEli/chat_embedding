# chat_embedding 

聊天机器人嵌入脚本

## Getting started
#### install

```shell
npm install chat_embedding
```

#### how to use

```JavaScript
import chatEmbedding from "chat_embedding"

chatEmbedding.run({
  // 聊天机器人服务地址
  serverUrl: "https://www.baidu.com/",
})

```
